# NodeJS 설치

`nvm` 다운로드 후 설치.  
nodeJS 설치하면 `npm`, `yarn` 사용 가능함

```shell
# nvm 1.1.8 이상을 사용해야 Corepack을 지원함
# nvm 다운로드 후 설치할것

# nodeJS 최신 버전 설치
nvm install lastest
nvm use 18.3.0

node -v # nodeJS 18.3.0
npm -v # mpm 8.11.0
```
